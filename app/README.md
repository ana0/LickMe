# Vite App with Taquito (TypeScript)

This is a Vite application configured to work with Tezos using Taquito, written in TypeScript.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Features

- Connect to Tezos wallet using Beacon Wallet
- Get account balance
- Ready to interact with Tezos smart contracts

## Configuration

The app is configured to use the Ghostnet testnet by default. You can change the network in `src/main.js`.
