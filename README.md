# LickMe Project

This project contains three main directories for different purposes:

## Directory Structure

### 1. `Tezos/contracts`
A Tezos smart contract project using SmartPy.

**Setup:**
```bash
cd Tezos/contracts
pip install -r requirements.txt
python contract.py
```

See `Tezos/contracts/README.md` for more details.

### 2. `app`
A Vite-based web application using Taquito to interact with Tezos.

**Setup:**
```bash
cd app
npm install
npm run dev
```

See `app/README.md` for more details.

### 3. `display`
A simple bare HTML page with no framework dependencies.

**Usage:**
Simply open `display/index.html` in your web browser.

See `display/README.md` for more details.
